The C programming language is used often in operating systems and has influenced other languages for decades

The C programming language is a computer programming language that was developed to do system programming for the operating system UNIX and is an imperative programming language. C was developed in the early 1970s by Ken Thompson and Dennis Ritchie at Bell Labs. It is a procedural language, which means that people can write their programs as a series of step-by-step instructions. C is a compiled language.

    What is C?
  • General-purpose programming language
  • Very popular language, despite being old
  • Fundamental language in the field of computer science
  • Strongly associated with UNIX
  • Developed to write the UNIX operating system

    What can C do?
  • Can generate machine code/native code
  • Enablles code portability between different machines
  • Manipulate pointers, a function in memory
  • Write compilers or interpreters for other languages

    Why Learn C?
  • One of the most popular programming languages in the world
  • Will have no problem learning other popular programming languages
  • Will understand how computer memory works
  • Very fast, compared to other programming languages, like Java and Python
  • very versatile; it can be used in both applications and technologies
    C has influenced the development of many popular programming languages, such as C++, Java, and Python, and it remains widely used today. Below are interesting facts about C:
  • Dennis Ritchie at Bell Labs created C in the early 1970s as a systems programming language that is efficient and portable.
  • Developers have used C to create a wide range of software applications, including operating systems, compilers, and games.
  • C is known for its low-level programming capabilities, which allow developers to access and manipulate hardware directly.
